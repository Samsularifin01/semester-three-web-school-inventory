Aplikasi inventaris barang untuk SDN Pasarejo 01 — solusi sederhana, modern, dan mudah dipakai untuk mencatat, memantau, dan mengelola barang sekolah.

## ✨ Sorotan
- 🔎 Pencatatan barang lengkap (nama, jumlah, lokasi, kondisi)
- 🧾 Riwayat peminjaman dan pengembalian
- 📊 Laporan ringkas untuk pengelola sekolah
- 🔒 Otentikasi dasar untuk admin dan pengguna
- ⚡ UI modern, fokus pada kemudahan penggunaan

## 🛠️ Teknologi
- HTML / CSS / JavaScript (frontend)
- Backend pilihan: Node.js / PHP / Python (sesuaikan dengan implementasi)
- Penyimpanan: SQLite / MySQL (sesuaikan)

## ⚡ Cara Cepat Menjalankan (Quick Start)
1. Clone repo:
   ```
   git clone https://github.com/riziqhilmi/project.git
   ```
2. Masuk ke folder proyek dan ikuti petunjuk instalasi pada dokumentasi masing-masing stack:
   - Backend: install dependency & jalankan server
   - Database: jalankan migration/import data awal
3. Buka aplikasi di browser: `http://localhost:3000` (atau sesuai konfigurasi)
